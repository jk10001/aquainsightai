# Data licensing summary

This repository includes datasets under multiple licences (e.g., CC BY 4.0, CC BY-NC 3.0, CC BY-SA 4.0, Open Government Licence – Ontario).

Authoritative licence terms and attribution requirements are documented per dataset in:
`sample_datasets/<dataset-name>/README.md`.

Important: Some datasets may have NonCommercial (NC) or ShareAlike (SA) obligations. 
Users are responsible for complying with each dataset’s licence.
